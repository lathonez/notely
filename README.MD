## Notely - Angular 1.6.10 & Typescript 2.8.3 note taking companion - CD via Circle CI + GitHub Pages
[![CircleCI](https://circleci.com/gh/lathonez/notely.svg?style=shield)](https://circleci.com/gh/lathonez/notely)

[Live Demo](https://lathonez.com/notely)

### Install

`npm install`

### Start

`npm start`

### Test

`npm test`

### Release to Github Pages (live demo)

`npm run release`
